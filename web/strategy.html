Strategy & Roadmap

Our product direction and the technology that powers Autonomi.

Product Roadmap

Phase 1 — Protocol Genesis

Mainnet-ready Autonomi contract on Arc: collateralised borrowing, LTV monitoring, and agent-triggered rebalancing. Mock oracles and testnet deployment for validation.

Target: Q4 2024 / Q1 2025

Phase 2 — Governance & Monitoring

DAO governance for parameters (LTV caps, agent allowlist). Enhanced dashboard with live positions, agent activity log, and SMS-style alerts. Circle-managed wallets integration.

Target: Q1 2025

Phase 3 — Ecosystem & Scale

Multi-chain expansion via Arc. Stork oracle integration for production USYC price feeds. Institutional-grade analytics and risk dashboards.

Target: Q2 2025+

Tech Stack

Chain & Oracles

  • Arc — Execution layer and testnet
  • Stork — Price oracle (USYC feeds)
  • Circle — USDC, USYC, managed wallets

Smart Contracts

  • Solidity — Autonomi, MockTeller, MockUSYC
  • Foundry — Build, test, deploy
  • OpenZeppelin (Ownable, ReentrancyGuard)

Backend & Frontend

  • Node.js / TypeScript — Agent, cron, API
  • viem — Contract reads and writes
  • Static web (Tailwind, vanilla JS)

Built on Arc • Powered by Circle • Guarded by Stork
